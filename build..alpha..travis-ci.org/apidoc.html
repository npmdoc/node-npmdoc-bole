### bole.reset()

Clears all output streams from the application

### bole.setFastTime()

If speed is something you care about and you can handle time in milliseconds since epoch (`Date.now()`) rather than the full ISO
 string (`new Date().toISOString()`) in your logs then use `bole.setFastTime(true)` to shave off some precious microseconds.

Note that this will reset to the default of `false` when you use `bole.reset()`
